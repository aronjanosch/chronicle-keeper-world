directly there — replace a file (keep the name) to update the image on the landing page.

`site/assets/screenshots/` (`library.png`, `overview.png`, `codex.png`, `page.png`,
`atlas.jpeg`, `timeline.png`, `graph.png`, `search.png`, `keeper.png`, `session.png`).
